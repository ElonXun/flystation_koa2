## [RESTful API for flystation](https://github.com/ElonXun/flystation_alpha)

   该项目主要提供 RESTful API 接口 给开源博客系统 
   
   其基于nodeJs下koa2框架
  
   由[koa-generator](https://github.com/17koa/koa-generator)脚手架生成
      

## 技术栈
   ```
    koa2
    mongoose
   ```
